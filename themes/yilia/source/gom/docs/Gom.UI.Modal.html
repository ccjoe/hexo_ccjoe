Modal

Gom.UI. Modal

new Modal(opts)

弹层底层抽象类，如果需要自定义弹出层才需要用到, 自定义一般于Modal.layout，不满足才需要用到此类
弹层层的家庭比较大，有Loading，confirm， alert, center, popover, tips, popup, top , bottom, toast
所有弹出层不可共存，但modal与toast可一起显示.

Parameters:
Name Type Description
opts object

传入的opts参数，会覆盖static默认参数

Properties
Name Type Attributes Default Description
type string <optional>

自定义GomUI不存在的弹出层组件时才需要指定
title string <optional>

弹层标题
content html | string <optional>

弹层的html内部
class string <optional>

弹层自定义class
mask boolean <optional>
 false

弹层是否显示遮罩
close boolean <optional>

当且仅当为true时会在右上角显示关闭小图标
btns object <optional>

弹出层组件时按钮

Properties
Name Type Attributes Description
yes string <optional>

确定按钮名称
no string <optional>

取消按钮名称

refreshEvent(frag, env)

刷新events事件对象
解决三种情况：
1.当没有wrapper时，render()返回fragmentHTML,组件上的events对象无法绑定事件，当fragmentHTML插入document后，可以调用此方法重新绑定events对象事件
2.不在view组件的 wrapper里面，但想需要被 view组件的events对象解析绑定事件的html
3.组件实例化后新增或删除事件（还没有通过此方法实现，待考证是否有必要）

Parameters:

所在的选择器或元素
